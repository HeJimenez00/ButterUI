## How to install

<Steps>
  <Step title="Install Butter/ui" description="Install Butter/ui via npm to get access to all core components.">
    <TerminalCodeBlock code={`npm i @butter-js/ui`} />
  </Step>

  <Step title="Install Tailwindcss" description="To add Tailwind CSS, use the following <a class='external-link' href='https://docs.astro.build/en/guides/styling/#tailwind' target='_blank' rel='noopener noreferrer'>Astro command</a>.">
    <TerminalCodeBlock code={`npx astro add tailwind`} />
  </Step>

  <Step title="Install GSAP" description="Install GSAP to enable smooth and performant animations in Butter components." >
    <TerminalCodeBlock code={`npm i gsap`} />
  </Step>

  <Step title="Install CVA" description='CVA helps manage conditional classes using variants, making your components cleaner and easier to maintain.'>
    <TerminalCodeBlock code={`npm i class-variance-authority`} />
  </Step>
</Steps>
