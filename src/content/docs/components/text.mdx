## Features
1. <span>Character, word, and line animations</span>
2. <span>Staggered animation effects</span>
3. <span>No conflicting animations</span>
4. <span>Text masking support</span>
5. <span>Font-loading aware</span>
6. <span>Reusable with unique instances</span>

## API Reference

| Prop            | Default          | Type                          | Description                                                                                                       |
| --------------- | ---------------- | ----------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| `as`            | 'p'            | `string`                      | HTML element to render (`p`, `h1`, `span`, etc.).                                                                 |
| `textType`      | 'chars'        | `'chars' \| 'words' \| 'lines'` | Animation granularity.                                                                                            |
| `class`         | -             | `string`                      | Additional CSS classes to apply to the wrapper.                                                                  |
| `mask`          | 'false'        | `true \| false`             | Enables text masking effect.                                                                                     |
| `tweenFrom`     | {}             | `TweenAnimation`              | Initial GSAP config (see <ExternalLink href="https://greensock.com/docs/v3/GSAP/TweenVars">TweenConfig</ExternalLink>). |
| `duration`      | 0.7            | `number`                      | Animation duration in seconds.                                                                                   |
| `opacity`       | -              | `number`                      | Initial opacity (0–1).                                                                                            |
| `y`             | -              | `number`                      | Initial vertical offset in pixels.                                                                               |
| `x`             | -      | `number`                      | Initial horizontal offset in pixels.                                                                             |
| `delay`         | -      | `number`                      | Delay before animation starts, in seconds.                                                                        |
| `ease`          | 'power4.out'   | `string`                      | <ExternalLink href="https://greensock.com/docs/v3/Eases">GSAP easing</ExternalLink> type.                          |
| `staggerAmount` | 0.1            | `number`                      | Time (in seconds) between each element’s animation.                                                              |
| `staggerFrom`   | 'start'        | `string`                      | Starting point for stagger (see <ExternalLink href="https://greensock.com/docs/v3/GSAP/Timeline/stagger">stagger docs</ExternalLink>). |

| Prop            | Default       | Type                                                                 | Description                                                                                         |
| --------------- | ------------- | -------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| `as`            |'p'         | `string`                                                             | HTML element to render (e.g. `p`, `h1`, `a`, etc.).                                                 |

| `class`         | -          | `string`                                                             | Additional CSS classes to apply to the wrapper.                                                     |
